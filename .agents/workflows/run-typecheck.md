# Run Typecheck

When asked to run TypeScript type checking for Splitpay, follow these steps:

1. Ensure dependencies are installed:

   ```bash
   npm install
   ```

2. Run the typechecker:

   ```bash
   npm run typecheck
   ```

3. Fix any reported type errors in the relevant files, keeping TypeScript strictness and project conventions in mind.
